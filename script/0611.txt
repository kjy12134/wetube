1. babel은 최신 버전의 js (es6, es7)이 브라우저랑 호환이
안되는 문제가 많아서 표준 js 버전의 언어로 컴파일 해주는 프로그램

2. nodemon 은 원래 소스 변경하면 서버 재시작을 수기로 매번
해줘야 하는데 그런 번거로움 없이 소스 수정하고 저장하면 자동으로
서버 restart 해서 반영해주는 프로그램

3. middleware 대박... java의 인터셉터 같은 기능이라고
생각하면 됨. requesthandler고 실제 request 받아서
처리 하기 전에 체크하는 로직 (예를들면,, 로그 남기거나
세션 체크하거나 그런거) 넣어서 핸들러 역할을 해주는 것
- middleware는 여러개 만들수 있고 적용도 여러개 할 수 있다
- 작성한 순서에 따라 route별로 미들웨어를 실행할수도있고 안할수도 있음
