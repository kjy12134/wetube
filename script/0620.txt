arrow function
a1. function abc () {  
    return true;
}

a2. const abc = () => true;

- 동일한 함수, =>는 함수 내용을 암시적으로 return하는
기능을 갖고 있음. 따라서 return을 또 명시해주면 오류가나고
arrow 뒤에 중괄호를 사용하면 암시적 리턴 기능을 잃으므로
그땐 괄호 안에 return을 명시해줘야 한다.
